# 🏎️ Neon Drift: High-Fidelity Antigravity Racing Engine

![Neon Drift Banner](https://img.shields.io/badge/Status-Production--Ready-brightgreen)
![Tech Stack](https://img.shields.io/badge/Tech-Vanilla%20JS%20%7C%20HTML5%20Canvas-orange)

**Neon Drift** is a high-performance, sensory-overload racing experience built entirely in Vanilla JavaScript. Designed for the "Boss Level" enthusiast, it pushes the boundaries of browser-based graphics and audio synchronization.

## 🌟 Key Features
- **Neural-Sync Aesthetics:** High-fidelity neon visuals with glassmorphism UI elements.
- **Dynamic Gravity Swapping:** Low-latency mechanics for hyper-speed gameplay.
- **Procedural Audio:** Integrated Web Audio API for immersive, beat-synced soundscapes.
- **Mobile-First Optimization:** Fully responsive with custom touch controls and APK support.
- **Boss Mode Challenges:** Escalating difficulty with procedural obstacle generation.

## 🛠️ Technical Deep Dive
- **Engine:** Custom-built 2D physics engine optimized for 60FPS on mobile and desktop.
- **Rendering:** Direct Canvas manipulation for minimal overhead and maximum juice.
- **State Management:** Robust system handling transitions between menu, gameplay, and "System Failure" states.
